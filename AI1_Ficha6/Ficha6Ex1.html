Cascading Style Sheets (CSS)

CSS is used to style and layout web pages — for example, to alter the font, color, size, and spacing of the content, split it into multiple columns, or add animations and other decorative features.

O atributo background-image é usado para definir uma imagem de fundo para um elemento. Os valores permitidos incluem URLs de imagens, gradientes, ou até mesmo múltiplas imagens. Um exemplo de uso é background-image: url('imagem.jpg'); ou background-image: linear-gradient(to bottom, gray, white), onde um gradiente linear é aplicado como fundo.
